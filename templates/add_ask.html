How to Make a Post

This system uses a Rich Editor CKEditior. It allows users to edit and format text, adding Images to text. To make a post, fill in the required sections and add the necessary information, afterward, click on the submit button. If you choose to add an Image to your post to support it, simply click on the Image Icon and add the URL link to the post, adjust the size to fit into it. The maximum image size is “250 X 250”, anything other than that will be too large, and make sure to give an H and V spacing of “10” respectively, then finally, select the position of the Image, either Left or Right.

CKEditor is a rich text service provider met to make text formatting and editing easy, you can choose the header style, and Font using the Editor with special image adding features. However to use the Upload function in the CKEditor, please contact the service or user for more Information.
